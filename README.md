# Superstore_Sales_dataset
# Pivot Table Sales Analysis Dashboard

## Project Overview
This project demonstrates data analysis using Excel PivotTables. 
The objective is to analyze sales data, create interactive pivot tables and charts, 
and generate meaningful business insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Steps Performed
1. Cleaned the dataset and verified headers and data types.
2. Created multiple PivotTables to analyze Sales by Category, Region, and Segment.
3. Added a calculated column for Profit Margin (Profit / Sales).
4. Built PivotCharts to visualize sales distribution and compare performance.
5. Inserted slicers for Region, Category, and Segment to make the dashboard interactive.
6. Highlighted top-performing regions using conditional formatting.
7. Wrote key analytical insights based on PivotTables.
8. Exported the final pivot report as a PDF.

## Deliverables
- Pivot_Report.xlsx – Excel file with pivot tables, charts, and slicers.
- Pivot_Report.pdf – PDF version of the final dashboard.
- Insights.docx – Document containing summarized analytical insights.

## Key Insights
- Technology category is the highest contributor to overall sales.
- West region is the top-performing region.
- Consumer segment generates the maximum sales.
- Office Supplies category shows lower sales compared to others.
- Central region has relatively lower performance.
